# API Test

A simple Express.js API that responds to GET requests with "Hello, World!".

## Installation

```bash
npm install
```

## Running the Server

```bash
npm start
```

The server will run on port 3000 by default. You can access it at `http://localhost:3000`.

## API Endpoints

- `GET /` - Returns "Hello, World!"

## Development

```bash
npm run dev
```
